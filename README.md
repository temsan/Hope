# Галерея работ Надежды Александровны Терёшкиной

Веб-галерея в стиле fluid art с параллакс эффектами.

## Два варианта использования

### Вариант 1: Единый HTML файл (рекомендуется)

Генерирует один HTML файл со всеми изображениями, встроенными в base64.

1. Установите Node.js: https://nodejs.org/

2. Установите зависимости:
```bash
npm install
```

3. Сгенерируйте галерею:
```bash
npm run generate
```

4. Откройте созданный файл `gallery.html` в браузере

**Преимущества:**
- Один файл - легко распространять
- Работает без сервера
- Thumbnails для быстрой загрузки
- Фоновая подгрузка полноразмерных изображений
- Оптимизация и сжатие изображений

### Вариант 2: Динамический сервер

Загружает изображения из папки в реальном времени.

```bash
npm start
```

Откройте: http://localhost:3000

## Возможности

- Fluid art дизайн с анимированными градиентами
- Параллакс эффекты при скролле
- Интерактивные элементы, реагирующие на движение мыши
- Thumbnails для быстрой загрузки галереи
- Полноразмерные изображения по клику
- Фоновая предзагрузка полных изображений
- Навигация стрелками клавиатуры
- Адаптивный дизайн
